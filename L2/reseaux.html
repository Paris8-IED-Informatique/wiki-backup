Exercice 8.8

Attention, certains fournisseurs d'accès bloquent le port 25 qui est celui utilisé pour transmettre de serveur à serveur (ceci afin d'éviter le surplus de spams). Pour certains fournisseurs (en France), on peut désactiver le blocage (voir https://homeprotection.fr/content/65-comment-desactiver-le-blocage-smtp-sortant-sur-ma-box-internet-ou-modifier-le-filtre-smtp-port-25   ), pour d'autres c'est a priori impossible.
