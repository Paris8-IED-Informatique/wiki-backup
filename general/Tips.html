LaTeX - Taille de PDF

Si cela peut être utile à certains, bonne lecture…

Problématique : PNG trop volumineux

Dans mon rendu LaTeX, le PDF de rendu dépasse la taille autorisée lors de mon envoi via Thunderbird. 
J'ai donc une erreur d'envoi, et dois donc réduire la taille de ce PDF, en le compressant.

Plusieurs options s'offrent à moi, dont des services en ligne, tels que ilovepdf ou d'autres. Cependant, dans un souci de ne pas envoyer mes fichiers à une plateforme externe, je vais tenter de chercher des outils à utiliser en local.

Source de la taille du PDF

Avant de chercher quel outil pourrait m'être utile, il faut d'abord que je connaisse la source de mon problème.

Ayant pas mal d'image, je vais commencer par là, avec l'outil pdfimages, qui peut être installé avec le package poppler-utils.
Installation : 

> sudo apt install poppler-utils

Vérification du pdf : 

> pdfimages -list "not_compressed.pdf"

Je peux voir que ma problématique, dans ce cas-ci, est liée à la taille de mes images embarquées.

Compression des images

En utilisant l'outil pngquant (lorsqu'il s'agit de fichier png), car dans mon cas, je veux conserver une transparence de certaines images, on peut réduire la taille de nos images PNG : 

installation :

> sudo apt install pngquant

compression : 

> pngquant --quality=65-80 --speed 1 png_folder/*.png
> sudo mv png_folder/*-fs8* png_folder_compressed/
