GitHub Student Developer Pack

Le pack du développeur proposé par Github, indispensable à tout étudiant, permet de découvrir gratuitement de nombreux outils payants utilisés dans le monde professionnel, à l’aide de votre adresse mail de Paris 8 : https://education.github.com/pack

Modèle LaTeX pour les devoirs

Un modèle de document en LaTeX pour les devoirs. Il est adaptable pour les autres matières, évidemment, mais il faudra alors le modifier en fonction des consignes données.

https://github.com/amartos/TexIEDP8

Emacs pour les devoirs

Une version arrangée d'Emacs pour l'IED appelée “IDEmacs”, conçue pour faciliter la prise en main de cet environnement de développement intégré (IDE), qui s'avère plutôt pratique pour la rédaction des devoirs pour l'IED dans cette configuration.

https://github.com/AreTaro/iedmacs.git

Paramètre SMTP et IMAP pour le client messagerie

Pour ceux qui souhaite utiliser un client messagerie tierce (comme Thunderbird) : 

voici les paramètres SMTP IMAP (MAJ fonctionne sur thunderbird le 01/08/2023 par ATALAY Tuncay) :

IMAP : 
nom du serveur : imap.etud.univ-paris8.fr
nom d'utilisateur : prenom.nom@etud.univ-paris8.fr
sécurité de la connexion : SSL/TLS
méthode d'authentification : mot de passe normal
port : 993

SMTP :
smtp.etud.univ-paris8.fr
port : 587
sécurité de la connexion : STARTTLS
Méthode d'authentification : Mot de passe normal
Nom d'utilisateur : prenom.nom@etud.univ-paris8.fr
